# InterviewRecipe
개인이 원하는 대로 면접 주제와 질문을 만들어 연습할 수 있는 커스텀 모의 면접 웹 어플리케이션  

<br/>

# 기능
- 회원 기능
- 주제 및 질문 CRUD 기능
- 모의 면접 실행 기능(TTS, WebRTC)

  <br/>

# STACK
> FRONT
  >  - jsp, html, css, javaScript, jQuery, ajax, TTS, WebRTC

> BACK
  >  - springBoot, springSecurity, java, myBatis

> DATABASE
  >  - mySQL
